If you are logging in on a new device, or after a fresh install of the Edge app, you may encounter one of the screens below.  
  
This notice is informing users that they must wait a limited **Security Period** before their account is released for this new login attempt.  
  
Users may by-pass this security period at any time by entering a 2FA code (if this was setup prior to losing access) or by accepting the login attempt from another device which is still logged in.  
  
If your security period is longer than 14 days, you can appeal for a reduction, down to the minimum of 7-days.  
  
***If you would like to request a shorter security period, please use the Live Chat button in the below corner of this screen and answer the questions provided.***
