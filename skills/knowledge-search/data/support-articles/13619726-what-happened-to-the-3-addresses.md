Edge supports native segwit (bech32) addresses for BTC, DGB, GRS, LTC, VTC, UFO. This style of address utilizes a unique prefix which will help prevent users from sending cryptocurrencies to the wrong wallet, for instance, sending LTC to a BTC wallet. All segwit-enabled wallets will automatically be upgraded to show addresses in bech32 format by default. There is an option in the upper left three-dot menu to view the 3-address for compatibility with services that have not upgraded to the new format.

Note, even though your wallet is showing bech32 addresses Edge still derives and monitors all previous 3-addresses. Any funds sent to the old addresses will still be detected.
