-   # Loan Dashboard
    
    -   This scene allows you to see an overview of all the loans you have and can be used to start a new loan by tapping the "+ New Loan" option at the bottom of the screen.
        
-   # Loan Status
    
    -   This scene allows you to see the progress of a loan that you have started. Each step will fill as the loan process completes.
        
-   # Loan Details
    
    -   This scene provides a breakdown of individual loans. You can see the breakdown of tokens borrowed, total amount borrowed, and other important details.

-   # Close Loan
    
    -   This scene allows you to close a loan. NOTE: Closing your loan will liquidate some of the collateral to pay the remaining principle and interest of your loan.
        
-   # Withdraw Collateral
    
    -   This scene allows you to remove collateral for use elsewhere. Please note that you will need to keep your Loan to Value Ratio positive or risk liquidation.
        
-   # Add Collateral
    
    -   This scene allows you to add collateral to keep your loan in a healthy status.
        
-   # Make Payment
    
    -   This scene allows you to pay your loan balance.
        
-   # Borrow Details
    
    -   This scene shows the current interest rate and value of your loan.
        
-   # Borrow More
    
    -   This scene allows you to borrow additional funds.
        
-   # Borrow With Aave
    
    -   This scene is a breakdown of the loan you plan on taking with Aave before you accept the loan.
