**To add a Custom Token (ex. ERC20), simply follow the steps below:**

**1)** Tap "Assets" located within the bottom navigation bar.

**2)** Tap on the main (parent) asset and tap on the three vertical dots, then tap "Add/Edit Tokens"

\- This will bring you to the Manage Tokens screen  
​

Note: you can also enter the contract address into the search bar on the Assets screen

**3)** Tap on "Add Custom" and enter the details for this token

-   **On this screen you will need to enter the following:**
    
    -   **Token Name**
        
    -   **Token Symbol**
        
    -   **Contract Address**
        
    -   **Decimal Places**
        

This information can be found on [etherscan.io](https://etherscan.io/) or the respective block explorer by typing in the token's contract address. All of the information will be viewable under the "Profile Summary" section.

-   Once added make sure that it is checked in the Manage Token Screen
    
-   You will now see the newly added Token at the Wallets List Screen
    

You will be able to store, receive, and send that token out as you wish.

**If you would for Edge to add a new token, please fill out this form and we will review the request shortly!**

[New Token Request Form](https://form.asana.com/?k=0EM9Iy0RD5Qvd6aWsYeTRw&d=9976422036640)

*Edge reviews each token before it gets officially added to the list of assets.*

Note that you will need some of the parent coin to send out any ERC20 token. For more information on this topic please visit [Why do I need ETH to send Ethereum Tokens.](https://support.edge.app/en/support/solutions/articles/8000088420-why-do-i-need-eth-to-send-ethereum-tokens-)
