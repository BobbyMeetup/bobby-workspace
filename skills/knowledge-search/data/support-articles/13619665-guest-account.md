Edge has been at the forefront of providing users with a secure and user-friendly experience in managing their cryptocurrencies. Traditionally, users have been required to create an account with a username, password and PIN to ensure the security of their digital assets. However, Edge has introduced a new and convenient feature called **Guest Account**, allowing users to streamline the onboarding process and start using the app almost instantly.

# Guest Account Setup

Unlike the conventional account creation process, the Guest Account setup in Edge is simplified. Users can install the app, set up a secure PIN, and start creating wallets without the need for a username or password.

## Backup Reminder

While the Guest Account provides users with immediate access to the app, Edge emphasizes the importance of account backup. Users are prompted with a warning message on the bottom of every scene and every time they tap on Receive or Buy/Deposit, reminding them to create a full account by creating a unique username and password. This serves as an essential backup measure, enabling users to recover their accounts in case of device loss or if they simply uninstall the app.  
A backup reminder warning card also appears if the user registers a FIO handle.

# Local Encryption for Enhanced Security

One of the key features of Edge is its commitment to user security and privacy. Once the user is ready to backup the account, **the user credentials are locally encrypted**. This means that the encryption process occurs on the user's device, and **the Edge servers never have access to the user's credentials**. This robust security measure ensures that user credentials remains private and protected.

## Limited Capabilities for Safety

To prioritize user safety, Edge has implemented restrictions for Guest Account users. Currently, users with Guest Accounts cannot deposit cryptocurrencies through the app and they can buy or sell with a max limit of 50 USD. This limitation ensures that users can explore and utilize the various features provided by Edge without exposing themselves to potential high risks associated with financial transactions.

## Comprehensive Access to Features

Despite the above limitations, users with Guest Accounts can access a range of other features provided by Edge. This includes the ability to create and manage wallets, import external wallets, monitor cryptocurrency balances, create FIO handles and explore the app's user-friendly interface. The Guest Account option allows users to experience the full spectrum of Edge's capabilities while maintaining a focus on security.

Edge's introduction of Guest Account creation represents a significant step towards enhancing user convenience in the realm of cryptocurrency management. By simplifying the onboarding process and encouraging the creation of a full working account, Edge ensures that users can enjoy a seamless and secure experience, making Edge a reliable choice for both novice and experienced cryptocurrency enthusiasts.
