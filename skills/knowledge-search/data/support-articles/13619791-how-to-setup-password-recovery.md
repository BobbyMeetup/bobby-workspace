After you have created your account, next you will need to set up security and Password recovery questions.

You will want to create a password recovery in case of forgotten password or device is compromised.

Once you are in your Edge account go to:

1.  Settings, located at the upper right.
    
2.  Once you have opened the setting, tap on Setup password recovery
    
3.  Enter your password.
    
4.  You will be asked to set up the answer to security questions
    

Once answered click done, you will then be asked to send the information to yourself via email or other storage method. Your password recovery token has been created and you will be able to login with this information if you forget your password.
