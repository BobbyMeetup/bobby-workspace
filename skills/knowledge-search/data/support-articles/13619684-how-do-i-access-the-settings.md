The Settings can be accessed by tapping on the 3 horizontal lines located at the top right corner. Tapping on that will bring out the Side Menu and below you will find the Settings.

Inside the Settings you will have the option to make a number of changes such as:

-   Change Password
    
-   Change PIN
    
-   2FA
    
-   Password Recovery
    
-   Exchange Settings
    
-   Auto Logout
    
-   Restore deleted wallets
    
-   And other options
    

Additionally each wallet has it's own menu. Tap into a wallet and you can then tap the three vertical dots located at the upper right to access the wallets menu.

This menu contains the following:

-   Split functions which allows you to move keys to different chains
    
-   Rename
    
-   Resync
    
-   Export Transactions
    
-   Master Private Key
    
-   Add/Edit Tokens
    
-   Get Raw Keys
    
-   Archive Wallet
