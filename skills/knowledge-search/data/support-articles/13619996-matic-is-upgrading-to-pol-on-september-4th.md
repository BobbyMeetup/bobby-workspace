# Key Points:

-   The MATIC → POL upgrade **will be automatic** for holders of MATIC within Edge.
    
-   POL will simply replace MATIC as the gas and staking token for Polygon.
    
-   Holders of MATIC *tokens* on Ethereum must be manually upgraded to POL; however, there is no deadline for this migration.
    
-   Holders of MATIC *tokens* on BNB Smart Chain will need to bridge tokens to Ethereum or exchange them for MATIC on Polygon before the upgrade.

# MATIC → POL

Following the Polygon Improvement Proposal ([PIP-17](https://forum.polygon.technology/t/pip-17-polygon-ecosystem-token-pol/12912)), the community has agreed on replacing the old MATIC ticker for POL. Initially, this is simply a change of the token's name, but there are plans to continue building more functionality as Polygon 2.0 advances their blockchain technology.

Edge users holding MATIC in a Polygon wallet will not need to do anything. Edge will update the ticker symbol in-app. Other self-custody wallets may require you to update RPC settings for Polygon, or the wallet may still display “MATIC” instead of “POL” as the token symbol.

# Re: MATIC Tokens

Edge is also supporting MATIC tokens on Ethereum and BNB Smart Chain. For Ethereum based MATIC tokens, you can upgrade the tokens using the official Polygon Portal Interface. More information on that process can be seen, here: [MATIC token migration for ETH and Polygon zkEVM (POL upgrade)](https://support.edge.app/hc/en-us/articles/28711643332379-MATIC-token-migration-for-ETH-and-Polygon-zkEVM-POL-upgrade)

For MATIC tokens on BNB Smart Chain, the easiest way to migrate would be through using the Exchange tab in Edge to trade these tokens for MATIC on Polygon to prepare for the switch. After MATIC fully transitions to POL, you can instead swap for this newer POL token, or manually upgrade the MATIC by bridging from Binance Smart Chain to Ethereum, and then using [Polygon's migration interface](https://portal.polygon.technology/pol-upgrade).
