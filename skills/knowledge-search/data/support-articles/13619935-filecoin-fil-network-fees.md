# Filecoin (FIL) Network – Powering Decentralized Storage

Filecoin is designed for storing and retrieving data across a decentralized network, but sending FIL tokens works just like sending crypto.

## How FIL Fees Work

-   **Base fee** for blockchain transaction
    
-   **Priority tip** for faster inclusion if needed  
    Fees are paid in **FIL tokens**.
    

Edge Wallet automatically calculates the best fee estimate for you based on current network congestion.

---

## Fee Options Available

-   **High** – Quickest confirmation
    
-   **Optimal (default)** – Balanced cost and speed
    
-   **Low** – Save on fees, wait slightly longer
    
-   **Custom** – Full manual control if desired
    

---

## Average Cost

Typically **$0.05 to $0.20 USD** per transaction depending on network activity.

---

## Why It Matters

Filecoin supports real-world decentralized storage and simple asset transfers. Edge Wallet optimizes FIL fees automatically so your transactions stay affordable and efficient.
