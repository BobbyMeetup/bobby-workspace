1.  Open Edge and tap "Assets" located within the bottom navigation bar.
    
2.  Tap on the wallet you want to review.
    
3.  Tap the transaction that you want the ID for.
    
4.  Inside the transaction, you will see the advanced details
    
5.  You can copy and paste the Transaction ID by tapping the copy and paste button or scroll down to "Show in Explorer" which will launch the transaction into Blockchair to view.
